# Chandan Kumar Portfolio

Modern personal portfolio showcasing projects, skills, achievements, and contact details. Built with React + Vite + TypeScript and a clean Tailwind UI.

## Live
- Portfolio: https://chandan785.netlify.app

## Highlights
- Responsive single-page layout
- Projects carousel with image gallery
- Achievements & certificates marquee
- Contact section with form
- Smooth animations with Framer Motion

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Local Setup
```sh
cd Portpholio
npm install
npm run dev
```

## Build
```sh
cd Portpholio
npm run build
```

## Optional: Contact Form (Supabase)
If you want the contact form to send emails through Supabase functions, add these environment variables to Portpholio/.env:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_anon_key
```
